## Автоматизированный пайплайн для сбора метрик смарт-контракта вольта типа ERC-4626.
- **Адрес волта:** 0x8ECC0B419dfe3AE197BC96f2a03636b5E1BE91db
- **Блокчейн:** Ethereum

## Функционал
- **Backfilling**: Сбор исторической информации по TVL и цене доли с момента деплоя контракта
- **Real-time Monitoring**: Постоянное отслеживание состояния вольта через Web3.py
- **Database**: Сохранение данных в PostgreSQL для последующего анализа
- **Security**: Использование переменных окружения (.env) для защиты API ключей.

## Как запустить
1. Склонируйте репозиторий.
2. Скопируйте `.env.example` в `.env` и подставьте свои ключи Alchemy/Postgres
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите скрипт: `python etl_with_history.py`
